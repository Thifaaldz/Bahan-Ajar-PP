# ğŸ”§ 2. Install WSL2

## 2.1 Buka PowerShell sebagai Administrator

1. Klik **Start**
2. Ketik: `PowerShell`
3. Klik kanan â†’ **Run as Administrator**

---

## 2.2 Install WSL (Metode Otomatis - Direkomendasikan)

```powershell
wsl --install
```

Perintah ini akan otomatis:

- Mengaktifkan Windows Subsystem for Linux
- Mengaktifkan Virtual Machine Platform
- Menginstall WSL2
- Menginstall Ubuntu default

Setelah selesai â†’ **Restart komputer**

---

## 2.3 Set Default WSL ke Versi 2

```powershell
wsl --set-default-version 2
```

Cek status:

```powershell
wsl --status
```

Cek daftar distro:

```powershell
wsl -l -v
```

# ğŸ‘‘ 5. Set Default User Menjadi Root

Secara default Ubuntu menggunakan user biasa.

Jika ingin login langsung sebagai root:

Jalankan di PowerShell:

```powershell
ubuntu config --default-user root
```

atau

```powershell
ubuntu.exe config --default-user root
```

Verifikasi:

```powershell
wsl
whoami
```

Output harus:

```
root
```
